## 소개
 - 항공편 예매 시스템입니다.
 - 로그인 기능을 통하여 관리자와 이용자의 접근 권한을 다르게 두었습니다.
 - 이용자들의 평점 시스템을 통하여 각 항공사의 랭킹을 표시해줍니다.
 - 각 항공편의 가격을 비교하여 보여줌으로써 이용자가 원하는 티켓을 선택할 수 있도록 도움을 줍니다.

## 기능
#### 공통
- 현재 등록된 비행기 목록 확인
- 회원가입

#### 관리자
- 항공편 생성
- 항공편 정보 변경
- 항공편 삭제

#### 이용자
- 티켓 예매
- 예매 정보 변경
- 예매 취소
- 이용했던 항공사 평점 부여
